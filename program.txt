1. User starts program
2. Initialize map
3. Prepopulate map with json data
4. Truncate json file
5. Start interactive shell
6. User inputs command - get <language> <filename> || save <language> <filename>
7. Parse command (only accept get || save, others will return an error)
8. Execute command:
	9. if it's save, save to the map
	9. If it's get retrieve the code inside filename
10. If command is exit, exit interactive shell
11. Serialize data inside map to json
12. Write serialized data to json file
